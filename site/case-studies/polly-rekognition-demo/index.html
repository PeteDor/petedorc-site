<div class="eyebrow">Case Study</div>

<h1 class="h1">Polly + Rekognition Serverless Demos</h1>

<p class="sub">Public demos using API Gateway + Lambda, protected by Cognito Hosted UI and AWS WAF rate limiting. No data stored; SSM Parameter Store for secrets; cost‑controlled.</p>

</section>

<section class="container layout">

<article class="content">

<div class="card">

<h2>Context</h2>

<p>Goal: showcase AI/ML capabilities safely in public. Requirements: no PII retention, throttle abuse, quick rollback, and $&lt;5/month at low traffic.</p>

</div>

<div class="card">

<h2>Architecture</h2>

<ul>

<li>API Gateway (REST) → Lambda (Node/Python) → AWS SDK (Polly/Rekognition)</li>

<li>Cognito Hosted UI for sign‑in; Amplify Auth for client demo; WAF per‑IP rate limits</li>

<li>Assets in S3; CloudFront if needed; logs to CloudWatch</li>

</ul>

<figure class="card" style="padding:0"><img src="/assets/diagrams/polly-rekognition-demo.png" alt="Serverless architecture: API GW + Lambda with Cognito and WAF" style="max-width:100%;border-radius:14px"/><figcaptionclass="kbd" style="padding:12px">Diagram placeholder.</figcaption></figure>

</div>

<div class="card">

<h2>Decisions & Trade‑offs</h2>

<ul>

<li><strong>No persistent store:</strong> avoids privacy risk; limits analytics depth.</li>

<li><strong>WAF vs custom throttling:</strong> WAF is simpler and cheaper to operate.</li>

<li><strong>Hosted UI:</strong> fast to stand up, less customizable than a bespoke front‑end.</li>

</ul>

</div>

<div class="card">

<h2>Results</h2>

<div class="metrics"><span class="metric">Cost ≈ $&lt;5/mo</span><span class="metric">P95 480ms → 95ms</span><span class="metric">0 incidents</span></div>

</div>

<div class="card">

<h2>Cost & Ops</h2>

<ul>

<li>Lambda + API GW free/low tier; WAF adds ~$5/mo baseline.</li>

<li>SSM for secrets; rotations via script; alarms on 4XX/5XX surge.</li>

</ul>

</div>

<div class="card">

<h2>Improvements</h2>

<ul>

<li>Add CloudFront for global demo latency.</li>

<li>Introduce per‑user quotas via Cognito groups.</li>

<li>Bundle a CDK app for single‑command deployment.</li>

</ul>

</div>
